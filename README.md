This package contains a recommended set of lints for Flutter apps, packages, and plugins to
encourage good coding practices.

## Install

Add `carapacik_lints` as dev dependency to your `pubspec.yaml`.

```yaml
dev_dependencies:
  carapacik_lints: ^3.10.0
```

Create an `analysis_options.yaml` file at the root of the project with the following content:

```yaml
include: package:carapacik_lints/core.yaml
```
